# MtAw2e_Catalog
Spell catalogue and casting aid for Mage: The Awakening 2nd Edition.

This application is an alpha prototype module intended to eventually be integrated into the CofD NPC project.
At present the spells.json included in the source includes a high amount of custom spells and modifications based on a personal game of MtA2e I am participating in.
Deleting this file and remaking all the spells by copying them from the source book is the only way to return to vanilla.
